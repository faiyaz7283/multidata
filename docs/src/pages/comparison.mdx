# Comparison

One of the benefits of storing alternate values is the built-in functionality of comparison methods. These methods enable easy comparison 
and calculation of differences between main and alternate values.

## et()

Get 'equal to' comparison for the specified key(s). (main == alt)

<Properties>
  <Property name="this" type="object">
    A data2d or a data3d object.
  </Property>
  <Property name="key" type="string">
    A key identifier. This parameter is only available with data2d object; not available for data3d.
  </Property>
  <Property name="primaryKey" type="string">
    A data3d primary key. This parameter is only available with data3d object; not available for data2d.
  </Property>
  <Property name="dataKey" type="string">
    A data3d data key. This parameter is only available with data3d object; not available for data2d.
  </Property>
  <Property name="returns" type="bool">
    A boolean value.
  </Property>
</Properties>

## timestampChange()

Get segmented time difference for the specified key(s). (main - alt)

<Properties>
  <Property name="this" type="object">
    A data2d or a data3d object.
  </Property>
  <Property name="key" type="string">
    A key identifier. This parameter is only available with data2d object; not available for data3d.
  </Property>
  <Property name="primaryKey" type="string">
    A data3d primary key. This parameter is only available with data3d object; not available for data2d.
  </Property>
  <Property name="dataKey" type="string">
    A data3d data key. This parameter is only available with data3d object; not available for data2d.
  </Property>
  <Property name="years" type="bool">
    Switch to turn on/off the years segment. Off by default. _Optional. The default is false._
  </Property>
  <Property name="months" type="bool">
    Switch to turn on/off the months segment. Off by default. _Optional. The default is false._
  </Property>
  <Property name="weeks" type="bool">
    Switch to turn on/off the weeks segment. Off by default. _Optional. The default is false._
  </Property>
  <Property name="days" type="bool">
    Switch to turn on/off the days segment. On by default. _Optional. The default is true._
  </Property>
  <Property name="hours" type="bool">
    Switch to turn on/off the hours segment. On by default. _Optional. The default is true._
  </Property>
  <Property name="minutes" type="bool">
    Switch to turn on/off the minutes segment. On by default. _Optional. The default is true._
  </Property>
  <Property name="seconds" type="bool">
    Switch to turn on/off the seconds segment. On by default. _Optional. The default is true._
  </Property>
  <Property name="returns" type="string">
    A string value.
  </Property>
</Properties>
